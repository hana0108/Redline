# Backend Redline

API FastAPI para el alcance actual de Redline.

## Módulos activos

- autenticación
- roles y usuarios
- sucursales
- vehículos e imágenes
- clientes
- ventas
- configuración general
- reportes

## Módulos retirados del alcance

- SaaS / multi-dealer
- leads
- reservas
- portal público

## Rutas principales

- `/api/v1/auth/*`
- `/api/v1/roles/*`
- `/api/v1/users/*`
- `/api/v1/branches/*`
- `/api/v1/vehicles/*`
- `/api/v1/clients/*`
- `/api/v1/sales/*`
- `/api/v1/settings`
- `/api/v1/reports/*`
